# 🎵 TuneFlow: Music Analytics & Recommendation System

![Version](https://img.shields.io/badge/version-2.0.12-brightgreen)
![Platform](https://img.shields.io/badge/platform-Windows-blue)
![Framework](https://img.shields.io/badge/framework-.NET%208.0-blueviolet)

**TuneFlow** — это высокотехнологичное настольное приложение для управления музыкальной библиотекой, построенное на базе C# и WinForms. Проект сочетает в себе современный интерфейс в стиле Spotify, мощные инструменты аналитики данных и интеллектуальный алгоритм рекомендаций.

Разработано в рамках курсового проекта по дисциплине «Объектно-ориентированное программирование».

---

## ✨ Основные возможности

### 📂 Управление библиотекой
* **База данных:** Обработка 1000 музыкальных треков с характеристиками: название, исполнитель, жанр, год выпуска, длительность и рейтинг.
* **Умный поиск:** Мгновенная фильтрация по нескольким параметрам одновременно (название, артист, жанр) прямо в процессе ввода.
* **Сортировка:** Многокритериальная сортировка (включая обратный порядок) с визуальной индикацией активной колонки (▲/▼).

### 🧠 Рекомендательная система
* **AI-Engine:** Алгоритм подбора похожих композиций на основе весовых коэффициентов сходства жанров, временных периодов и пользовательских рейтингов.
* **Избранное:** Создание персональных плейлистов с сохранением состояния и визуальной маркировкой треков (★).

### 📊 Аналитический дашборд
* **Визуализация GDI+:** Кастомная отрисовка графиков без использования сторонних библиотек.
* **Гистограмма:** Распределение популярности по всем жанрам библиотеки.
* **Линейный график трендов:** Динамика количества выпусков или среднего рейтинга по годам с возможностью интерактивного переключения.
* **KPI-метрики:** Автоматический расчет общего тайминга библиотеки, средних показателей и топ-листов артистов.

### 🎨 Современный UI/UX
* **Spotify Dark Theme:** Глубокая темная палитра для комфортной работы.
* **Custom UI Components:** Полностью отрисованный вручную скроллбар и безрамочное управление окном.
* **High DPI Support:** Идеальная четкость шрифтов и элементов на современных мониторах.

## 🛠 Технологический стек

* **Язык программирования:** C# 12.0
* **Платформа:** .NET 8.0 (Windows Forms)
* **Графика:** GDI+ (System.Drawing) для отрисовки Dashboard
* **Архитектура:** Layered Architecture (Logic / UI separation)
* **Технологии:** 
    * **LINQ** — для высокоскоростной обработки данных.
    * **WinAPI (P/Invoke)** — для интеграции с Windows DWM и реализации перемещения безрамочного окна.
    * **Reflection** — для универсальной логики сортировки объектов.
